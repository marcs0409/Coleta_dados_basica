# Estrutura básica de coleta de dados

Este repositório contém um conjunto de scripts e exemplos para coleta de dados de forma automatizada. A estrutura é projetada para facilitar a implementação de novas coletas e a manutenção do código existente.

## Como usar

Url deve ser ajustada para o site desejado.
Ajuste também, no 'Pandas', quantas linhas deseja ler.

BeautifulSoup e requests apenas como exemplos.
